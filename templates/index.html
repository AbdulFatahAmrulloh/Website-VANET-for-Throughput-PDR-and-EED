About the Metrics
Throughput

Measures the rate of successful data transfer over a communication channel, expressed in Megabits per second (Mbps). Higher throughput generally indicates better network performance.

Packet Delivery Ratio (PDR)

The ratio of successfully delivered data packets to destination nodes compared to the number of packets sent by source nodes. Expressed as a percentage, with higher values indicating better reliability.

End-to-End Delay (EED)

The time taken for a packet to be transmitted across a network from source to destination, measured in milliseconds (ms). Lower values indicate better network responsiveness.
